A terminal sticky notes & to-do app.

## Install

```bash
brew tap 1Verona/terminoted
brew install terminoted
```

## Usage

```bash
terminoted
```

### Keybindings

| Key | Action |
|-----|--------|
| `tab` | Switch between To-Do and Notes tabs |
| `a` | Add new task / note |
| `enter` | Open a note |
| `x` | Toggle task done/undone |
| `d` | Delete task / note |
| `esc` | Save & close note editor |
| `q` | Quit |

## Data

Your data is stored in `~/.terminoted/`.
